# 🐶콩고물 토오크 v3.1

SillyTavern용 캐릭터 기반 보조 메신저 확장입니다.

## 무엇을 하는 확장인가요?

🐶콩고물 토오크는 현재 선택된 캐릭터와 별도의 작은 메신저 창에서 대화할 수 있게 해주는 확장입니다.
본채팅에 대화를 자동으로 넣지 않으면서도, 현재 캐릭터 카드와 페르소나, 최근 본채팅 맥락을 참고해 캐릭터와 따로 문자하듯 대화하는 것을 목표로 합니다.

## 주요 기능

- SillyTavern 확장 메뉴에서 🐶콩고물 토오크 창 열기
- 현재 캐릭터별로 대화방 저장
- 대화방마다 모드 고정
- 대화방 새로 만들기, 이름 변경, 방 삭제
- 말풍선 우클릭 또는 길게 눌러 복사 / 본RP 삽입 / 삭제
- 최대 응답 토큰 수, 최근 본채팅 읽을 메시지 수, 폰트 크기, 창 크기 설정
- 캐릭터 말투 고정 메모 지원
- 본채팅에는 자동 삽입하지 않음

## 모드

- **Care**: 가벼운 일상 대화, 기분 이야기, 소소한 고민, 정서/멘탈 관련 대화용 모드입니다. 상담 모드가 아니라 캐릭터가 실제로 할 법한 반응을 우선합니다.
- **Secretary**: 일정, 할 일, 정리, 우선순위, 간단한 판단 보조용 모드입니다. 캐릭터가 비서 역할을 맡아 유용하게 답하려고 노력하되, 모르는 분야는 캐릭터답게 헷갈리거나 추측할 수 있습니다.
- **Co-worker**: 같은 회사 동료 설정으로 실제 업무, 고객 대응, 마케팅, 문구, 제품 설명 등을 함께 보는 모드입니다. 같은 팀 동료처럼 이해하고 의견을 주되, 외부 컨설턴트처럼 말하지 않는 것을 목표로 합니다.
- **Watching RP**: 본채팅의 지난 장면과 흐름을 같이 보고 이야기하는 모드입니다. 장면 감상, 감정선 해석, 다음 전개 대화, 답변 보조에 사용할 수 있습니다.

## 설치

1. ZIP 압축을 풉니다.
2. GitHub 저장소 루트에 아래 파일을 업로드합니다.
   - `manifest.json`
   - `index.js`
   - `style.css`
   - `README.md`
3. SillyTavern에서 `Extensions → Install Extension`에 GitHub 저장소 주소를 넣어 설치합니다.
4. 설치 후 새로고침하거나 확장을 업데이트합니다.

## 업데이트 메모

### v3.1

- 네 가지 모드 모두 “본채팅 진행은 멈춘 상태의 별도 메신저 대화”라는 공통 전제를 강화했습니다.
- Care를 가벼운 일상 대화와 정서/멘탈 관련 주제를 다루는 모드로 재정리했습니다. Care가 자동으로 다정한 상담 말투가 되지 않고, 캐릭터가 할 법한 반응을 우선하도록 수정했습니다.
- Secretary는 캐릭터가 비서 역할을 맡아 유용하게 답하려고 노력하되, 캐릭터가 모를 법한 분야에서는 모르는 티가 나거나 조심스럽게 추측할 수 있도록 조정했습니다.
- Co-worker는 같은 회사 동료로서 실제 업무를 같이 보는 설정을 강화했습니다. 전문가/컨설턴트 말투보다 캐릭터다운 동료 반응을 우선합니다.
- Watching RP는 “롤플/RP”라는 메타 표현보다 본채팅의 지난 장면과 흐름을 같이 보는 방식으로 다루도록 수정했습니다.
- 캐릭터를 인위적으로 더 현명하거나 다정하게 보정하라는 식으로 읽힐 수 있는 문구를 제거하고, 최근 본채팅 말투와 캐릭터 카드의 성향을 더 직접적으로 따르도록 정리했습니다.

## v2.5 변경점
- 요청하지 않은 미래 약속, 직접 만남, 음식/선물/보상 제안, “내일 내가 해줄게”식 엔딩을 더 강하게 제한했습니다.
- Care 모드에서 정서 케어가 귀여운 보상/돌봄 약속으로 새지 않고, 현재 문자 대화 안에서 끝나도록 조정했습니다.


## v3.1 변경점

- 본채팅의 캐릭터 말투와 성격을 최우선으로 반영하도록 프롬프트를 다시 강화했습니다.
- Care/Secretary/Co-worker에서도 모드는 역할만 바꾸고 캐릭터성은 유지하도록 조정했습니다.
- 요청 없는 미래 약속, 직접 만남, 선물/음식/방문 제안이 나오지 않도록 공통 경계를 다시 넣었습니다.
- 콩고물 내부 이전 답변을 흉내내라는 보조 문구를 제거했습니다.
- 팝업 창을 헤더 드래그로 이동할 수 있게 수정했습니다.
