# 제목 미정 Assistant v0.7

SillyTavern용 별도 캐릭터 Assistant 메신저 확장입니다.

## v0.7 변경점

- 현재 대화방 이름 탭을 누르면 대화방 목록/내역을 열고 닫습니다.
- 삼선 버튼을 제거해 UI를 단순화했습니다.
- `+` 버튼을 누르면 새 대화방을 만들고 즉시 그 방으로 이동합니다. 기존 대화방은 삭제되지 않습니다.
- 모드별 프롬프트를 강화했습니다.
  - Lover / Care: 현실 고민, 일상 궁금증, 감정 정리, 멘탈 케어. 제4의 벽을 부수지 않습니다.
  - Secretary: 간단한 질문 답변, 정리, 일정, 할 일, 판단 보조.
  - 업무 동료: 사용자의 실제 업무를 도와주는 모드.
  - OOC 대화: RP 상황 해석, 감정선, 전개, 답변 구상 등을 돕는 RP 보조 모드.
- 모든 모드는 캐릭터 정체성, 말투, 성격, 관계성, 메모리를 유지하도록 지시합니다.

## 설치

GitHub 저장소 루트에 아래 파일들이 바로 위치해야 합니다.

- `manifest.json`
- `index.js`
- `style.css`
- `README.md`

SillyTavern에서 Extensions → Install Extension에 저장소 URL을 넣어 설치합니다.
