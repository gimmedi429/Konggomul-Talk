# 제목 미정 Assistant v0.4

SillyTavern 본 RP 채팅과 분리된 캐릭터 기반 Assistant 메신저 확장입니다.

## 핵심
- 현재 캐릭터 카드, 페르소나, 최근 본채팅 메시지를 읽어 별도 Assistant 대화에 반영합니다.
- 본 RP 채팅에는 자동으로 주입하지 않습니다.
- RP용 프리셋/서사 프리셋은 복사하지 않고, 확장 내부 프롬프트로 별도 생성합니다.
- 캐릭터별로 Assistant 대화방이 저장됩니다.

## 모드
- Lover / Care: 제4의 벽을 부수지 않고, 캐릭터/관계/메모리를 유지한 채 현실 고민·일상·멘탈 대화.
- Secretary: 캐릭터가 비서 역할로 정리·일정·할 일·판단 보조.
- 업무 동료: 같은 회사 동료 역할로 업무 질문·문구·CS·마케팅 도움.
- OOC 대화: 본 RP 상황을 제3자 관점으로 분석·정리·전개 보조.

## v0.4 변경
- 설정 대부분을 Assistant 창 안의 ⚙로 이동.
- 창 크기 직접 리사이즈 + 너비/높이 설정 추가.
- 대화방 드롭다운 제거, 깔끔한 대화방 목록 패널로 변경.
- 새 대화방 + 버튼 동작 개선.
- Connection Profile 목록 새로고침/선택 UI 추가.

## 주의
Connection Profile 선택은 SillyTavern의 `/profile-list`, `/profile` 슬래시 커맨드를 이용하는 방식입니다.
사용 중인 SillyTavern 버전/확장 상태에 따라 실패할 수 있으며, 그 경우 현재 선택된 ST 연결을 사용하세요.
