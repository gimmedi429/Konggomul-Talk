# 콩고물 스위치 v1.0

SillyTavern용 별도 캐릭터 메신저 확장입니다. 본 RP 채팅에 자동으로 입력되지 않고, 현재 캐릭터와 최근 본채팅을 참고해 별도 대화방에서 대화합니다.

## v1.0 변경점

- 확장 이름을 `콩고물 스위치`로 변경했습니다.
- 대화방마다 모드가 고정되도록 변경했습니다. 새 대화방은 현재 선택된 모드로 생성되고, 이후 해당 방의 모드로 유지됩니다.
- 모드 이름과 프롬프트를 수정했습니다.
  - Care: 현실 고민, 일상 궁금증, 감정 정리, 멘탈 케어 중심. 제4의 벽을 부수지 않습니다.
  - Secretary: 간단한 질문 답변, 정리, 일정, 할 일, 우선순위, 판단 보조.
  - Co-worker: 같은 회사 동료 설정으로 사용자의 실제 업무를 함께 보고 돕는 모드.
  - Watching RP: 본채팅을 함께 읽으며 장면, 감정선, 전개, 답변 구상, 설정 정리 등에 대해 대화하는 모드.
- 모든 모드는 캐릭터 정체성, 말투, 성격, 관계성, 메모리를 유지하도록 지시합니다.

## 설치

GitHub 저장소 루트에 아래 파일들이 바로 위치해야 합니다.

- `manifest.json`
- `index.js`
- `style.css`
- `README.md`

SillyTavern에서 Extensions → Install Extension에 저장소 URL을 넣어 설치합니다.
