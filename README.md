# 🐶콩고물 토오크 v3.9.4

SillyTavern용 캐릭터 기반 보조 메신저 확장입니다.

## 무엇을 하는 확장인가요?

🐶콩고물 토오크는 현재 선택된 캐릭터와 별도의 작은 메신저 창에서 대화할 수 있게 해주는 확장입니다.
본채팅에 대화를 자동으로 넣지 않으면서도, 현재 캐릭터 카드와 페르소나, 최근 본채팅 맥락을 참고해 캐릭터와 따로 문자하듯 대화하는 것을 목표로 합니다.

## 주요 기능

- SillyTavern 확장 메뉴에서 🐶콩고물 토오크 창 열기
- 현재 캐릭터별로 대화방 저장
- 대화방마다 모드 고정
- 대화방 새로 만들기, 이름 변경, 방 삭제
- 자주 쓰는 대화방 상단 고정
- 채팅 팝업 접기/펼치기
- 말풍선 우클릭 또는 길게 눌러 복사 / 본RP 삽입 / 삭제
- 최대 응답 토큰 수, 최근 본채팅 읽을 메시지 수, 폰트 크기, 창 크기 설정
- 캐릭터 말투 고정 메모 지원
- Co-worker 업무 메모 지원
- 현재 캐릭터 대화 내보내기 / 가져오기
- 활성화된 로어북(World Info) 내용 반영
- 본채팅에는 자동 삽입하지 않음

## 모드

- **Care**: 가벼운 일상 대화, 기분 이야기, 소소한 고민, 정서/멘탈 관련 대화용 모드입니다. 상담 모드가 아니라 캐릭터가 실제로 할 법한 반응을 우선합니다.
- **Secretary**: 해당 대화방에서 캐릭터가 유저의 개인 비서 역할을 맡는 모드입니다. 캐릭터가 황당해하거나 싫어하거나 재밌어할 수는 있지만, 일정·할 일·정리·우선순위·판단 보조 요청에는 성실하게 답하도록 설계했습니다.
- **Co-worker**: 해당 대화방에서 캐릭터가 유저와 같은 회사/팀의 동료가 되는 모드입니다. Co-worker 업무 메모를 참고해 실제 업무를 돕되, 캐릭터의 세계관과 말투를 유지하고 외부 컨설턴트처럼 말하지 않는 것을 목표로 합니다.
- **Watching RP**: 본채팅의 지난 장면과 흐름을 같이 보고 이야기하는 모드입니다. 장면 감상, 감정선 해석, 다음 전개 대화, 답변 보조에 사용할 수 있습니다.

## 설치

1. ZIP 압축을 풉니다.
2. GitHub 저장소 루트에 아래 파일을 업로드합니다.
   - `manifest.json`
   - `index.js`
   - `style.css`
   - `README.md`
3. SillyTavern에서 `Extensions → Install Extension`에 GitHub 저장소 주소를 넣어 설치합니다.
4. 설치 후 새로고침하거나 확장을 업데이트합니다.



### v3.9.8

- 캐릭터 선톡 트리거를 시간 기반에서 본채팅 캐릭터 답변 수 기반으로 변경했습니다.
- 선톡은 키워드/내용 감지가 아니라 답변 개수만 카운트합니다.
- 높음은 본채팅 캐릭터 답변 10개마다, 보통은 20개마다, 낮음은 30개마다 50% 확률로 선톡을 시도합니다.
- 선톡 내용은 Care 모드의 랜덤 가벼운 주제로 생성되며, 본채팅의 특정 단어를 분석하거나 요약하지 않도록 조정했습니다.

## v3.8 긴급 수정

- v3.1 기반으로 롤백 후, AI 연결 프로필 선택 방식을 다시 정리했습니다.
- 설정의 `AI 연결 프로필`에서 `저장된 Connection Profile 선택`을 고르고, 오른쪽 새로고침 버튼을 눌러 프로필 목록을 불러온 뒤 선택할 수 있습니다.
- 별도 프로필을 선택하지 않으면 현재 SillyTavern 메인 연결을 그대로 사용합니다.
- 콩고물 토오크 생성 중 선택 프로필로 임시 전환한 뒤, 생성이 끝나면 기존 본채팅 프로필로 복구하도록 보강했습니다.
- 본채팅 프로필을 확인할 수 없는 경우에는 메인 프로필 보호를 위해 생성을 중단합니다.

※ SillyTavern 내부에서 별도 Connection Profile을 완전히 독립 실행하는 공개 API가 안정적으로 보장되어 있지는 않아, 현재 방식은 “임시 전환 후 즉시 복구” 구조입니다. 사용 후 본채팅 프로필이 맞게 돌아왔는지 첫 테스트 때 확인해 주세요.
