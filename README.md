# 제목 미정 Assistant v0.5

SillyTavern 본 RP 채팅과 분리된 캐릭터 기반 Assistant 문자/카톡형 확장입니다.

## 핵심
- 현재 캐릭터 카드, 페르소나, 최근 본채팅 메시지를 읽어 별도 Assistant 대화에 반영합니다.
- 본 RP 채팅에는 자동으로 주입하지 않습니다.
- RP용 프리셋/서사 프리셋은 복사하지 않고, 확장 내부 프롬프트로 별도 생성합니다.
- 캐릭터별로 Assistant 대화방이 저장됩니다.
- 답변은 한국어를 기본으로 합니다.

## 모드
- Lover / Care: 제4의 벽을 부수지 않고, 캐릭터/관계/메모리가 유지된 상태로 현실 고민·일상 궁금증·멘탈 관련 대화를 나눕니다.
- Secretary: 같은 캐릭터가 비서 역할로 정리·일정·할 일·판단 보조를 합니다.
- 업무 동료: 같은 회사 동료 역할로 업무 질문·문구·CS·마케팅 도움을 줍니다.
- OOC 대화: 본 RP 상황을 분석·정리·전개 보조하는 RP 어시스턴트 모드입니다.

## v0.5 변경
- 왼쪽 확장탭의 버튼 제거: `확장 활성화 / 창 열기` 체크가 곧 창 열기입니다.
- v0.4에서 생긴 빈 대화방이 여러 개면 자동으로 1개만 남기도록 정리합니다.
- Assistant 창 안 ⚙ 설정에서 토큰 수, 최근 메시지 수, 폰트 크기, 창 크기를 조정합니다.
- 콩가루 인절미 컨셉의 밝은 베이지 CSS로 변경했습니다.
- 캐릭터와 문자/카톡을 주고받는 설정을 프롬프트에 강화했습니다.
- 현재 캐릭터의 Assistant 대화를 전부 초기화하는 버튼을 추가했습니다.

## 주의
Connection Profile 선택은 SillyTavern의 `/profile-list`, `/profile` 슬래시 커맨드를 이용하는 실험적 방식입니다.
사용 중인 SillyTavern 버전/확장 상태에 따라 실패할 수 있으며, 그 경우 현재 선택된 ST 연결을 사용하세요.
