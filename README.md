# 🐕콩고물 톡

SillyTavern에서 현재 선택된 캐릭터와 별도의 작은 메신저 창으로 대화할 수 있는 확장입니다. 본RP를 자동으로 이어 쓰지 않고, 캐릭터 카드·페르소나·최근 본채팅·활성화된 로어북을 참고해 캐릭터와 따로 문자하듯 대화합니다.

상단바에서는 짧게 **콩톡**으로 표시됩니다.

## 주요 기능

- 현재 캐릭터와 별도 메신저 대화
- 현재 캐릭터별 대화방 저장
- 대화방별 모드 영구 고정
- 새 톡방 생성 시 모드 선택
- 모드별 첫 메시지 자동 시작
- 대화방 이름 변경, 삭제, 상단 고정
- 채팅 팝업 접기/펼치기
- 접힌 버튼 드래그 이동
- 메시지 복사 / RP에 반영 / 삭제
- 최근 콩고물 톡 10건 요약 후 본RP 입력창에 삽입
- 최대 응답 토큰 수 설정
- 최근 본채팅 읽을 메시지 수 설정
- 채팅창 폰트 크기 설정
- 창 너비/높이 설정
- 캐릭터 말투 고정 메모
- 직장 동료 업무 메모
- 현재 캐릭터 대화 내보내기 / 가져오기
- 활성화된 로어북(World Info) 반영
- 별도 Connection Profile 선택 지원

## 모드

새 대화방을 만들 때 모드를 선택합니다. 선택한 모드는 해당 방에 계속 고정됩니다.

### 콩톡

캐릭터와 가볍게 문자를 주고받는 기본 톡방입니다. 일상 대화, 질문, 잡담, 투덜거림, 소소한 고민 등에 사용할 수 있습니다.

새 방 생성 시 첫 메시지는 `🐕` 하나로 시작합니다.

### 집사 모드

캐릭터가 유저의 집사 역할을 맡는 톡방입니다. 질문 답변, 일정 정리, 할 일 정리, 선택지 비교, 정보 요약, 판단 보조, 생활 보조 등에 사용할 수 있습니다.

새 방 생성 시 캐릭터가 집사 역할 선언문을 읽고, 자기 성격대로 반응한 뒤 대화를 시작합니다.

### 펫 모드

캐릭터가 유저의 펫 역할을 맡는 톡방입니다. 캐릭터의 성격, 자존심, 애정 표현 방식, 말투는 유지되며, 펫 역할에 대한 반응도 캐릭터 성격에 따라 달라집니다.

새 방 생성 시 캐릭터가 펫 역할 선언문을 읽고, 자기 성격대로 반응한 뒤 대화를 시작합니다.

### 직장 동료 모드

캐릭터가 유저와 같은 회사/팀의 직장 동료가 되는 톡방입니다. 업무 대화, 문구 작성, 일정 정리, 자료 정리, 아이디어 회의, 고객/학부모/외부 응대 문안, 업무 우선순위 정리 등에 사용할 수 있습니다.

새 방 생성 시 캐릭터가 직장 동료 역할 선언문을 읽고, 자기 성격대로 반응한 뒤 대화를 시작합니다.

### RP 어시 모드

진행 중인 RP를 옆에서 함께 보고 정리하는 톡방입니다. 상황 설명, 감정선 정리, OOC 문장 작성, 답변 방향 제안, 원하는 전개로 가기 위한 대화 초안 작성 등에 사용할 수 있습니다.

새 방 생성 시 캐릭터가 RP 어시 역할 선언문을 읽고, 자기 성격대로 반응한 뒤 대화를 시작합니다.

## RP에 반영

메시지를 우클릭하거나 길게 누르면 메뉴가 열립니다.

**RP에 반영**을 선택하면 최근 콩고물 톡 10건을 요약해 본RP 입력창에 삽입할 수 있습니다. 확인창에서 취소하거나 진행할 수 있습니다.

삽입되는 내용은 대화 전문 복사가 아니라, 본RP에 참고할 수 있는 요약 메모 형태입니다.

## 설정

확장 창의 설정 버튼에서 아래 항목을 조정할 수 있습니다.

- AI 연결 프로필
- 최대 응답 토큰 수
- 최근 본채팅 읽을 메시지 수
- 채팅창 폰트 크기
- 캐릭터 말투 고정 메모
- 직장 동료 업무 메모
- 창 너비
- 창 높이
- 현재 캐릭터 대화 내보내기
- 현재 캐릭터 대화 가져오기
- 현재 캐릭터 대화 전체 초기화

## 설치

1. ZIP 압축을 풉니다.
2. GitHub 저장소 루트에 아래 파일을 업로드합니다.
   - `manifest.json`
   - `index.js`
   - `style.css`
   - `README.md`
3. SillyTavern에서 `Extensions → Install Extension`에 GitHub 저장소 주소를 넣어 설치합니다.
4. 설치 후 새로고침하거나 확장을 업데이트합니다.

## 파일 구성

- `manifest.json`: 확장 정보
- `index.js`: 확장 기능
- `style.css`: 확장 UI 스타일
- `README.md`: 사용 설명
